Spark

SparkScala

  • Installation

    Install Spark 2.3

  • Setup shell environment

    Edit ~/.bash_profile or ~/.zshrc with

spark_shell_environment

export SPARK_HOME=.../spark-2.3.x-bin-hadoop2.x export PATH=$PATH:$SPARK_HOME:$SPARK_HOME/bin %SPARK_HOME%\bin #For Windows, add this to `envrionment variable`
  • Set log level

    Edit conf/log4j.properties with

    log4j.rootCategory=ERROR, console

    Setup with the following levels: ALL, DEBUG, ERROR, FATAL, INFO, OFF, TRACE, TRACE_INT, WARN

setup_log_info_level

// set logger import org.apache.log4j.{Level, Logger} Logger.getLogger("org").setLevel(Level.ERROR) Logger.getLogger("akka").setLevel(Level.ERROR) Logger.info(s"<infomation>")
