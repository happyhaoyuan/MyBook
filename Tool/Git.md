# Git

* Initialize
```bash
git init
```

* Add file
```bash
git add <files>
```

* Commit
```bash
git commit -m "<you commit>"
```

* Add the local to a remote repository
```bash
git remote add origin <your remote git ssh address>
```
* Push the change
```bash
git push (-u) (-f) origin master
```

* Check all branches
```bash
git branch -a
```

* Create a branch
```bash
git checkout -b <branch_name>
```

* Delete a branch
```bash
git branch -d <branch_name>
```

* Switch to a branch
```bash
git checkout <branch_name>
```

* Check the conflict
```bash
git status
```
> or

	```bash
	git log
	```
